Export scene to one/multiple URDF files or return URDF data as string(s).

Parameters:

  • fname (str) ‚Äì URDF filename or None. If None, the URDF data is returned as one or multiple strings.

  • folder (str, optional) ‚Äì Only used if fname is None. The folder in which to export the URDF. Affects the location of textures. Defaults to None.

  • mesh_dir (str, optional) ‚Äì Mesh directory to write polygon meshes to. Defaults to the directory part of fname or folder if fname is None.

  • use_absolute_mesh_paths (bool, optional) ‚Äì If set to True will convert all mesh paths to absolute ones. Defaults to False.

  • mesh_path_prefix (str, optional) ‚Äì Can be used to set file name prefix for meshes to ‚Äúfile://‚Äù or ‚Äúpackage://‚Äù. Defaults to ‚Äú‚Äù.

  • write_mesh_files (bool, optional) ‚Äì If False, mesh file names in the URDF will point to the asset source files. If True, the meshes will be written to the URDF mesh directory. Defaults to True.

  • write_mesh_file_type (str, optional) ‚Äì File type that will be used if write_mesh_files is True. Defaults to ‚Äúobj‚Äù.

  • separate_assets (bool, optional) ‚Äì If True, each asset in the scene will be exported to a separate URDF file, named according to its object identifier. Note: The scene itself will not be exported; each asset‚Äôs transformation will not be preserved. Defaults to False.

  • include_camera_node (bool, optional) ‚Äì Whether to include a link for the camera node. Defaults to False.

  • include_light_nodes (bool, optional) ‚Äì Whether to include a link for the light nodes. Defaults to False.

  • ignore_layers (bool, optional) ‚Äì If True will add all geometries as visual and collision geometries. If None, will it will be (num_layers < 2). Defaults to None.

If fname is None, will return the URDF data as a string. dict[str, str] (optional): If fname=None and separate_assets=True will return the additional assets as URDF strings in a dictionary.
